# 宁德时代（300750）研究笔记

**标的**：宁德时代 300750  
**行业**：动力电池 / 新能源  
**数据来源**：公开资料整理（学习用）

## 业务概况

- 全球动力电池龙头，客户覆盖主流车企
- 储能业务为第二增长曲线
- 技术路线涵盖磷酸铁锂、三元等

## 财务特征（定性）

- 营收规模大，但行业竞争加剧压制利润率
- 资本开支较高，需跟踪产能利用率
- 研发投入占比高，技术迭代快

## 主要风险

- 新能源车销量波动
- 二线电池厂价格战
- 原材料（锂、钴等）价格波动
- 海外贸易政策不确定性

## 研究结论（笔记）

成长逻辑清晰，但属于**周期+成长**双重属性标的，需关注装机量增速与单位盈利趋势。
