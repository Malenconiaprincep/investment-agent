# 贵州茅台（600519）研究笔记

**标的**：贵州茅台 600519  
**行业**：白酒 / 食品饮料  
**数据来源**：公开资料整理（学习用）

## 业务概况

- 高端白酒龙头，主打飞天茅台及系列酒
- 直销渠道占比提升，渠道改革持续推进
- 品牌护城河深，定价权较强

## 财务特征（定性）

- 毛利率长期维持高位，现金流充沛
- ROE 处于 A 股顶尖水平
- 预收账款/合同负债是观察景气度的辅助指标

## 主要风险

- 宏观经济与消费场景波动
- 估值中枢下移风险
- 政策与舆论对高端消费的扰动

## 研究结论（笔记）

适合作为**质量型白马**观察标的，需结合估值分位与批价走势综合判断，不宜仅凭品牌溢价追高。
