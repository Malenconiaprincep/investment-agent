# 银行板块研究笔记

**板块**：A 股银行  
**代表标的**：平安银行 000001、招商银行 600036  
**数据来源**：公开资料整理（学习用）

## 板块逻辑

- 银行盈利与宏观经济、利率环境高度相关
- 净息差是核心盈利驱动，资产质量（不良率）是关键风险指标
- 国有大行偏稳健，股份行与城商行弹性更大

## 观察指标

- 净息差趋势
- 不良贷款率 / 拨备覆盖率
- 零售 vs 对公贷款结构
- 股息率（部分投资者作为高股息配置）

## 主要风险

- 房地产相关资产质量
- 利率下行压缩息差
- 经济增速不及预期

## 研究结论（笔记）

银行板块适合**防御+股息**思路研究，个股分化大，不宜用单一指标（如 PE）横向硬比。
